# Js Midi Sequencer

This project is a web-based midi sequencer, allowing users to create, edit, and play musical patterns directly in their browser. 

## Features

*   **Step-based Sequencing:** Create musical patterns by activating steps in a grid.
*   **MIDI Input Support:** Connect a MIDI device to input notes and control parameters.
*   **Pattern Management:** Create, save, load, and delete musical patterns.
*   **Track Controls:** Adjust parameters like volume, pan, and more for each track.
*   **Responsive Design:** Works seamlessly on various screen sizes and devices.
*   **Real-time Playback:** Hear your creations play back instantly as you edit.

## Getting Started

```bash
    npm run dev
```
